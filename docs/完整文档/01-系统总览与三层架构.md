# 01 系统总览与三层架构

## 1.1 系统组成
HJSE-1001 网球训练馆系统由三个软件层级 + 硬件层构成：
- **小程序层**（`3 软件/tennisyy`）：uni-app(Vue2)，BLE 蓝牙控制发球机，含课程管理、发球控制、固件 OTA。
- **下位机固件层**（`2 电气/2.3 程序代码`）：STM32F103 电机驱动程序 V3，控制 6 路电机、角度闭环、故障检测。
- **Windows 程序层**（`TennisTraining/`）：.NET8/WinForms，集成视觉识别、发球机控制、投影显示、数据记录、系统协调五大模块。
- **硬件层**：高速摄像头、发球机(电机/控制板)、投影机、传感器。

## 1.2 三层关系与数据流
```
┌──────────────┐   BLE GATT 写    ┌──────────┐  UART2 透传   ┌──────────────┐
│  小程序       │ ──────────────► │ BLE 模块  │ ────────────► │  STM32 固件   │
│ (tennisyy)   │                 │(AT 命令初始化)│  Modbus RTU │ (ModBus_unpack)│
└──────┬───────┘  ◄──Notify 上行 └──────────┘ ◄──────────── └──────┬───────┘
       │ 同一套 Modbus RTU 协议                              BL_Read_DATA │
       │ (CRC16/寄存器/球参数编码 三方一致)                  → 电机驱动   │
       │                                                              │
┌──────▼──────────────────────────────────────────────────────────────▼───┐
│                        Windows 程序 (TennisTraining)                     │
│  串口/BLE ──► LauncherControlEngine ──► Modbus RTU 帧 ──► 同一固件       │
│  摄像头 ──► Vision ──► Coordination(状态机/编排) ──► Projection/Data     │
└──────────────────────────────────────────────────────────────────────────┘
```
- 小程序与 Windows 程序共用同一套 Modbus RTU 协议，均经 BLE/串口与同一固件对接。
- 固件 `Bluetooth.h` 的 `struct BL_Read` 是 App↔固件的寄存器映射，与小程序命令一一对应。

## 1.3 协议一致性（✓已验证）
| 项 | 小程序 | STM32 固件 | Windows 程序 |
|---|---|---|---|
| CRC16 | `util.CRCMB16`（表驱动，低字节在前）| `modbus.h ModBus_unpack` | `ModbusCrc16`（位式 0xA001）|
| 标准向量 | — | — | "123456789"→0x4B37 ✓ |
| 球参数编码 | `util.processBalls` | `Ball_SPIN` 0~4上/5平/6~10下 | `BallParameterEncoder` |
| 设备控制 | `changeDeviceState` 0x06@0x0003 | `BL_Read.STATE` 0/1/2 | `LauncherCommands` |
| 故障码 | `getErrorCodeTable` 0001~0008 | `Error_data`/`ERROR_WRITE` | `LauncherErrorCode` 1~8 |

## 1.4 运行环境
- 小程序：HBuilderX/微信开发者工具，Vue2，uni-app 编译。
- 固件：Keil MDK，STM32F103 标准外设库，J-Link 烧录。
- Windows：.NET 8 SDK + Windows Desktop 运行时，`dotnet build`/`dotnet run`。
- BLE：BLE 模块经 UART2(115200) 接 STM32，AT 命令初始化广播名 `HJ-<设备ID>`。

## 1.5 主要参数范围
- 球速：固件 `Ball_Speed` km/h；小程序 speed 字段；映射字节 0~255。
- 球频：1~10 球/3s（`Ball_FREQ`）。
- 旋转：`Ball_SPIN` 0~4 上旋 / 5 平击 / 6~10 下旋。
- 水平角度：`Ball_ANGLE2` 0~90° 对应 -45°~+45°。
- 上下角度：`Ball_ANGLE1` 0~90°。
- 目标数：`Target_NUM`，达到后自动停机。
- 电量：`BATTERY` 0~100%。
