# 04 小程序接口完整说明（tennisyy / uni-app）

> 源码：`3 软件/tennisyy`。Vue2 + uni-app，BLE 蓝牙控制。

## 4.1 工程与入口
- 入口：`App.vue`(globalData+eventBus)、`main.js`、`pages.json`、`manifest.json`(appid `__UNI__5D49D50`)。
- globalData 关键：BLE UUID/特征、`aucCRCHi/aucCRCLo`(CRC 表)、`bleConnectionStatus`、各 storage 键名、`course_version`。
- eventBus：`on/emit/off`，跨页通信(如 `onBledata` OTA 回传)。

## 4.2 页面结构（pages.json）
主包 `pages/`：
- `index/index` 首页；`index/DeviceManager` 设备管理(连接/解绑/信号)；`index/upgrade`、`chooseFileUpgrade` OTA；`index/stuckBall` 堵球；`index/QrCodeScan` 扫码；`index/updateLesson` 课程更新。
- `devPosition/devPosition` 摆放位(1-6)；`lessonSetting/lessonSetting` 课程参数；`lessonList/lessonList` 课程列表；`serveControl/serveControl` 发球控制(核心)；`serveControl/countdownDialog` 倒计时；`instructions/instructions` 说明。
分包 `pages_two/`：`addLesson` 新增、`lessonChange` 修改、`lessonShow`/`userLessonShow` 展示、`view/CustomSlider`、`view/TimePicker`。

## 4.3 BLE 连接流程（utils/bleUtil.js）
`connectBleDevice` → `wx.createBLEConnection`(30s 超时) → Android `uni.setBLEMTU(247)` → `getBLEDeviceServices`(匹配 Service `55535343-...`) → `getBLEDeviceCharacteristics`(绑定读写/Notify 特征) → `notifyBLECharacteristicsId`(使能 Notify)。写：`bleWrite(str)`→`string2Buffer`→`writeBLECharacteristicValue`(失败重试3次)。收：`onBLECharacteristicValueChange`→`buf2hex`→`util.checkCrc`。

## 4.4 命令构造（serveControl.vue）— 见 02 协议手册
- `changeDeviceState(flag)`：`01060003`+值+CRC（0停/1开/2故障/3暂停）。
- 微调：`0106001D`+值+CRC（1弧度+/2弧度-/3水平+/4水平-/5球频+/6球频-）。
- `goModel()`：`0110001A 0005 0A`+10字节数据+CRC（按1组/2计时/3自由模式填充）。
- `sendAllBalls()`：`util.buildSendList(finalBalls)` 生成3包，每包15球；按 `flag_order` 1顺序/2随机(`shuffleArray`)；发完调 `changeDeviceState(1)` 启动。
- 帧间延时 `num_send_timeout`(约20ms)；蓝牙断开视为断电，重连后 `flag_device_state=0` 需重发全部课程数据。

## 4.5 课程数据模型（data/courseData*.js）
6 个摆放位各一文件(courseData1..6)，结构 `{version, lessonData, lessonDataTwo, lessonDataMore}`。
课程对象：`name`、`devPosition`(1-6)、`isCollect`、`currentDiff`(1/2/3)、`diffOne/diffTwo/diffThree`(三档球参数数组)。
球参数：`spinType`(1上旋/2平击/3下旋)、`leftRight`(-15..15)、`speed`、`rate`(球频)、`height`(弧度)、`hand`(1左/2右)、`spin`(上下旋强度)。
`utils/lesssonDataUtils.js`：`initLessonToStorage`(首次写 storage)、`getLessonDataById`、`getCollectedData`、`checkUpdateLessons`/`updateLessonData`(按 `course_version` 升级，保留 `isCollect`)。

## 4.6 本地存储键（globalData）
`save_ble_name`、`save_user_lesson`(自定义)、`save_diff`、`save_favorite`、`save_lesson_one..six`(6摆放位内置)、`save_loop_group_count`、`save_loop_group_time`、`save_serve_mode`(1组/2计时/3自由)、`save_serving_order`(1顺序/2随机)、`save_mode_group_num`、`save_mode_time_num`、`save_course_version`、`save_quick_type/id`、`bleDevicesCacheKey`。

## 4.7 固件 OTA（upgrade.vue / chooseFileUpgrade.vue）
bin→base64(`openssl base64 -A`)，按 `sliceSize` 分片，自定义帧 `sendPackageInfoCode`(包信息+sliceCount+CRC)+`sendPackageSliceCode`(index+len+分片+CRC)；`queryVersionCode` 查版本、`startUpgradeModeCode` 进升级模式；`bom.json` 管理版本号。

## 4.8 故障码（bleUtil.getErrorCodeTable）
0001 上发球电机堵转 / 0002 下发球电机堵转 / 0003 拨球电机堵转 / 0004 进球口堵球 / 0005 旋转电机堵转 / 0006 旋转电机故障 / 0007 俯仰电机堵转 / 0008 俯仰电机故障。

## 4.9 工具函数（util.js / ku.js）
`CRCMB16`(表驱动 Modbus CRC16)、`buildSendList`(球数据分包)、`processBalls`(球参数映射)、`shuffleArray`(随机)、`decimalToHex`/`padHexToLength`(十六进制补齐)、`buffer2hex`/`string2Buffer`、`checkCrc`、`compareVersion`、`delay`。
